# 缠论解盘 - 善缘版 Windows

---

### 善缘版是基于 Chanlun-Pro 项目开发的免费桌面程序；

### 最新版本：V1.0.1  

_更新时间：2022-12-02_

扫码入QQ群，在群文件中进行下载使用；

![QQ](img/qq.png)

行情数据只包括沪深A股市场；

缠论计算与图表展示只支持固定配置，不可更改；

可添加删除自选，没有自选组功能；（双击可增加/删除自选）

内置线段三买点选股功能，需要手动触发执行；

线段三买选股每日只能执行一次，建议15点收盘后执行；
